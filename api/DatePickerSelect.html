Class DatePickerSelect - extends Calendar

A base class for DatePickerSelect, providing:
  • Widget Lifecycle (initializer, renderUI, bindUI, syncUI, destructor)
  • Select a date from Calendar to select elements
Quick Example:
var instance = new A.DatePickerSelect({
displayBoundingBox: '#displayBoundingBoxId',
// locale: 'pt-br',
dateFormat: '%m/%d/%y',
yearRange: [ 1970, 2009 ]
}).render();

Configuration Attributes

appendOrder - Array

The order the selects elements are appended to the displayBoundingBox.
Default Value: [ 'm', 'd', 'y' ]

baseName - String

A basename to identify the select elements from this DatePickerSelect.
Default Value: datepicker

dayField - {Node | String}

HTML element to receive the day value when a date is selected.
Default Value: Generated HTML select element

dayFieldName - String

Name attribute used on the dayField.
Default Value: day

displayBoundingBox - {Node | String}

The container boundingBox to house the selects and button. The boundingBox attribute is used on the Calendar Overlay.
Default Value: null

monthField - {Node | String}

HTML element to receive the month value when a date is selected.
Default Value: Generated HTML select element

monthFieldName - String

Name attribute used on the monthField.
Default Value: month

populateDay - boolean

If true the select element for the days will be automatic populated.
Default Value: true

populateMonth - boolean

If true the select element for the month will be automatic populated.
Default Value: true

populateYear - boolean

If true the select element for the year will be automatic populated.
Default Value: true

setValue - boolean

Inherited from Calendar.
Default Value: false

trigger - {Node | String}

Trigger element to open the calendar. Inherited from OverlayContext.
Default Value: Generated HTLM div element

visible - boolean

If true the Calendar is visible by default after the render phase. Inherited from OverlayContext.
Default Value: false

yearField - {Node | String}

HTML element to receive the year value when a date is selected.
Default Value: Generated HTML select element

yearFieldName - String

Name attribute used on the yearField.
Default Value: year

yearRange - Array

Year range to be displayed on the year select element. By default it displays from -10 to +10 years from the current year.
Default Value: [ year - 10, year + 10 ]
